# NodeJS Pagination Scraper Example

This repository provides an example of how to scrape paginated data from a website using:
- Node.js
- [node-fetch](https://www.npmjs.com/package/node-fetch) (for HTTP requests)
- Simple regex parsing for `<tr>` and `<td>` elements

---

## Setup

1. **Clone or download** this repository.
2. **Install Node.js** (version 14 or higher recommended).
3. **Navigate** into the project folder and install dependencies:
   ```bash
   npm init -y
   npm install node-fetch@2
